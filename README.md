# Binary Search Tree (BST) - Inorder Traversal

## Overview
This project implements a Binary Search Tree (BST) in Python and performs an inorder traversal.

## Inorder Traversal
Order: Left → Root → Right

This prints values in sorted order for a BST.

## How It Works
- Insert places values in correct BST position
- Inorder traversal recursively visits left, root, right

## Output Example
For inputs: 5, 2, 7, 10, 4, 1

Output:
1
2
4
5
7
10
